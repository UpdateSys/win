How to use

Four simple steps

From copy to activation in under a minute.

01

Copy the main command

Click the copy button on the PowerShell terminal above.

02

Run the TLS command first on old Windows builds

Required for Windows 7, 8.1, and Server 2012 to enable TLS 1.2.

03

Open PowerShell as Administrator

Right-click the Start menu and select "Windows Terminal (Admin)" or "PowerShell (Admin)".

04

Paste the command and press Enter

Right-click to paste, then press Enter. The activator will launch automatically.
